Workspace Deployer- Windows Version

Move shortcut to desktop or wherever.
DO NOT MOVE run.exe from the folder that contains the "jre" folder. Create a shortcut and place that anywhere you'd like.

Persisted data will be stored in -
	C:\Users\<username>\Documents\foofinc\WSD_AppData/startups.ser

Creating startup paths-
	URLs - copy and paste URL (literally can't mess it up)
	Executable files (.exe, .sln, or really any file that can be double clicked)- Copy absolute path (C:\...\<filename>.exe)
